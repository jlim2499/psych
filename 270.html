What is the positive predictive value of the new test?


19/39

The correct answer is 19/39. The positive predictive value (PPV) of a test is the probability that subjects with a positive screening test truly have the disease. It is calculated as the number of true positives (those with the disease who test positive) divided by the total number of positive tests results, both true and false positives. In this case, there are 19 true positives (men with prostate cancer who tested positive) and 20 false positives (men without prostate cancer who tested positive), so PPV = 19/(19+20) = 19/39.

The first option, 19/20, would be the sensitivity of the new test if all men with prostate cancer were to test positive. Sensitivity is defined as the proportion of actual positives that are correctly identified as such. However, in this case, not all men with prostate cancer tested positive - some tested negative - hence this option is incorrect.

The second option, 723/743, would represent the specificity of the new test if all men without prostate cancer were to test negative. Specificity refers to the ability of a test to correctly identify those without the disease. Here though, not all men without prostate cancer tested negative - some tested positive - making this option incorrect.

The fourth option, 19/33, seems to miscalculate PPV by adding up false negatives (men with prostate cancer who tested negative) and true positives (men with prostate cancer who tested positive). This does not provide an accurate measure for PPV.

Lastly, 723/737 incorrectly calculates PPV by dividing true negatives (men without prostate cancer who tested negative) by total negatives which includes both true negatives and false negatives.

Therefore, none of these options apart from 19/39 accurately calculate the Positive Predictive Value in this scenario.
