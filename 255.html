Which of the following correctly describes the purpose of descriptive statistics?

To present a characteristic features of a data set

The correct answer is To present a characteristic features of a data set. Descriptive statistics summarise and describe the features of a dataset, giving a clear overview of the dataâ€™s distribution, central tendency, and variability through measures such as the mean, median, mode, range, and standard deviation. This allows for a concise presentation of large amounts of data for easy understanding and comparison.

To determine if a sample accurately describes the population from which it is drawn is incorrect. This is typically the realm of inferential statistics, which use samples to make inferences or predictions about a population.

To determine whether two population means are equal is incorrect. This is also a task for inferential statistics, specifically hypothesis testing, rather than descriptive statistics.

To make an estimation of a population mean from a sample is not the primary purpose of descriptive statistics. This is part of inferential statistics, which estimate population parameters based on sample statistics.

To test hypotheses based on sample data is incorrect. Hypothesis testing is a function of inferential statistics, not descriptive statistics.
