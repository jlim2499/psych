A college professor collects the results of a test from a group of students. Each test then receives a grade from A to F (with A being the best and F the worst). Which of the following would be the most appropriate measure to describe the most common mark?

Mode

The median would give the middle result, i.e. if there were 7 results as follows A, B, C, D, E, F, F then the median result would be a D. The question asks for the most common result however for which the mode would be used. In the above example the modal value would be an F. The mean would not be an appropriate measure in this case as the sum of (A + B + C + D + E + F + F) / 7 would be a meaningless result.
