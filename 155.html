In an attempt to better understand the religious needs of her patients, a psychiatrist creates a Likert scale questionnaire. The questionnaire is then given to 100 community patients.
The data is collected and is found not to be skewed. Which of the following is true?


The mode would be an appropriate measure of central tendency for this data set

Likert scales are ordinal scales. A Likert scale offers respondents a number of options to each question. For example:

Question:

How often do you pray?

Answer:

Never
Rarely
Occasionally
Frequently
Daily

Likert scales produce ordinal data sets, for this the mode and median are preferred as summary measures. The mean does not yield a meaningful result and so although used by some, it must be used with caution as it requires a number of assumptions to be stated to assist with interpretation. In view of this it cannot be stated that it is the 'most appropriate' summary measure.

The standard deviation should not be used for ordinal data.
