In evaluating concurrent validity for a new psychometric test for PTSD, which of the following approaches would be most suitable?

Administering the new test alongside a well-validated PTSD measure

The correct answer is Administering the new test alongside a well-validated PTSD measure. Concurrent validity is established by comparing the results of a new psychometric test with those of an existing, well-validated measure that assesses the same construct. If the new test's scores strongly correlate with the established tool, it demonstrates that the new test accurately measures PTSD at the same point in time. This approach directly evaluates whether the new test is valid for its intended purpose.

Examining the ability of the test to predict long-term functional outcomes: This approach assesses predictive validity, not concurrent validity. Predictive validity focuses on how well the test can forecast future outcomes, which is distinct from evaluating its accuracy at the same time as another established measure.

Testing whether it differentiates PTSD from generalised anxiety disorder: This approach evaluates discriminant validity, which examines whether a test can distinguish between different constructs. While this is an important psychometric property, it does not address concurrent validity.

Administering the test multiple times to assess its internal consistency: This approach is related to reliability, specifically test-retest reliability or internal consistency, which measures the consistency of the test results over time or across items. It does not assess whether the test aligns with another validated measure at the same time.

Correlating test scores with the duration of trauma exposure: This approach evaluates construct validity, which tests whether the measure aligns with theoretical expectations (e.g., higher PTSD symptoms with prolonged trauma exposure). While important, it does not assess the concurrent validity of the test against an established measure.
