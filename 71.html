A researcher hypothesises that a new drug will successfully treat depression. He completes a study which suggests his hypothesis was correct when in fact it was not. What has occurred?

Type I error

Type I error = rejecting a true null hypothesis
The null hypothesis in this case would be:

Ho = The drug does not lead to a change in depression (either severity or prevalence, it doesn't matter for the sake of the question).

If his hypothesis (that the drug treats depression) is incorrect then the null hypothesis (Ho) is true.

He has concluded that the drug is effective when it was not and so has rejected the null when it was true (the drug did not improve depression). This is a type I error.

Type I errors relate to finding differences that do not exist.
