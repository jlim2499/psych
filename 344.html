What is the correct formula to calculate the negative predictive value of a screening test?
TP = true positive; FP = false positive; TN = true negative; FN = false negative



TN / (TN + FN)

The correct answer is TN / (TN + FN). The negative predictive value (NPV) is the probability that a person with a negative test result truly does not have the disease. This is calculated by dividing the number of true negatives (TN) by the total number of people who tested negative (TN + FN). Clinically, the NPV is crucial in screening tests where ruling out a disease is essential, such as excluding serious conditions like sepsis in a paediatric emergency setting. If a test has a high NPV, clinicians can be more confident that patients with a negative result are unlikely to have the disease.

TP / (TP + FP) is the formula for positive predictive value (PPV), not NPV. PPV provides the likelihood that a person with a positive test result actually has the disease. While important for confirming diagnoses, it is not relevant to calculating NPV, which focuses on negative results.

TN / (TN + FP) is the formula for specificity, which measures the test's ability to correctly identify individuals without the disease. Specificity is useful for understanding the test's performance but does not address the probability that a negative test result is accurate (NPV).

Sensitivity / (1 - specificity) is the formula for the likelihood ratio positive (LR+), not for NPV. LR+ assesses how much more likely a positive test result is in someone with the disease compared to someone without it, which does not directly address the question of NPV.
