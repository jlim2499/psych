The following graph represents an ROC curve.

Which of the following would be represented on the y axis?


Sensitivity

The correct answer is Sensitivity. In an ROC (Receiver Operating Characteristic) curve, the y-axis represents sensitivity, while the x-axis represents 1 - specificity. Sensitivity, also known as true positive rate, measures the proportion of actual positives that are correctly identified as such. It is a measure of how well a test can identify true positives.

The option True negative rate is incorrect because this is represented by specificity and not sensitivity. Specificity measures the proportion of negatives which are correctly identified, and it would be plotted on the x axis as 1 - specificity.

The option 1 - Sensitivity is also incorrect. This would represent the false negative rate or Type II error (failing to reject a false null hypothesis). In an ROC curve, we plot sensitivity against 1 - specificity, not 1 - sensitivity.

False negative rate is another incorrect choice for similar reasons to '1 - Sensitivity'. The false negative rate refers to the likelihood of a test incorrectly classifying a positive case as negative. This value does not form part of an ROC curve's axes; rather it influences the shape and position of the curve within its axes.

Lastly, Sensitivity - Specificity is also not correct. An ROC curve plots sensitivity against 1- specificity, so subtracting these two values would not give you any meaningful measure in this context.

Remember that interpreting an ROC curve involves assessing how well a diagnostic test distinguishes between two states (e.g., disease present/absent), factoring in both its sensitivity and specificity at different threshold levels.
