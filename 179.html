A test is devised to assist in the diagnosis of a new psychiatric condition. The prevalence of the condition in the population is 4/11 (0.36 2dp). The following properties of the test are known:

Sensitivity = 0.8
Specificity = 0.4
Positive predictive value = 0.57 (2dp)
Negative predictive value = 0.67 (2dp)

Based on the above information select the most appropriate answers to the questions below

209.The pre-test odds
0.57 Pre-test odds = (pre-test prob / (1 - pre-test prob)
= (4/11) / (1 - (4/11)
= (4/11) / (7/11)
= 44/77
= 0.57 (2dp)


210.The likelihood ratio for a positive test
1.33 LR+ = sensitivity / (1 - specificity)
= 0.8 / (1 - 0.4)
= 1.33 (2dp)


211.The post-test odds
0.76 Post-test odds = pre-test odds x LR+
= 0.57 x 1.33
= 0.76 (2dp)
