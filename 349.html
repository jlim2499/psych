Which one of the following statements regarding significance tests is correct?

Chi-squared test is used to compare non-parametric data

The correct answer is Chi-squared test is used to compare non-parametric data.

The chi-squared test is a statistical test used to determine whether there is a significant association between categorical variables. It is appropriate for non-parametric data, such as frequencies or counts in different categories, rather than continuous data.
