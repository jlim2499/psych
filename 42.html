Which of the following is a fundamental criterion in Susser’s framework for causation?

Time order

Previous Exam Question

The correct answer is Time order. In Susser’s framework, time order is a fundamental criterion for causation. The cause must precede the effect in time. Without establishing a temporal relationship, causality cannot be inferred.

Strength: Strength is a criterion in Bradford Hill’s framework and refers to the magnitude of an association. It is not a foundational element of Susser’s framework.

Consistency: Consistency, part of Bradford Hill’s criteria, assesses whether an association is observed across different studies and populations. It is not a core requirement in Susser’s criteria.

Specificity: Specificity, also part of Bradford Hill’s criteria, evaluates whether a cause leads to a specific effect. Susser’s framework does not prioritise specificity as a criterion for causation.

Coherence: Coherence, a Bradford Hill criterion, ensures that the observed association aligns with existing biological or epidemiological knowledge. It is not a component of Susser’s criteria.
