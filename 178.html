Which of the following is true of correlation and regression?

Regression allows one variable to be predicted from another variable

The correct answer is Regression allows one variable to be predicted from another variable. Regression is a statistical method that helps in understanding the relationship between a dependent variable and one or more independent variables. It is particularly useful in making predictions. For example, in clinical practice, regression can be utilised to predict a patient's blood pressure based on their age, weight, and lifestyle factors.

If no correlation is found, regression is often still useful. This statement is incorrect. Correlation and regression are related concepts; a lack of correlation between variables suggests that there is no linear relationship to model through regression. If there is no correlation, regression analysis would not provide a meaningful predictive model. Clinically, this means that if two clinical measurements do not correlate, using one to predict the other would be unreliable.

Correlation is concerned with demonstrating difference between variables. This is not accurate. Correlation measures the strength and direction of a linear relationship between two variables, but it does not concern itself with differences between variables. It only indicates how closely related the variables are. Clinically, understanding correlation helps in identifying associations but does not imply causation or difference.

A t-test is a common test of correlation. This is incorrect. A t-test is used to compare the means of two groups and determine if they are statistically different from each other.
