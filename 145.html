A psychiatrist develops a new test to aid in the diagnosis of a condition. The condition has an estimated prevalence of 1/16. Given the following properties of the test, calculate the post-test probability for a positive test.
Sensitivity = 0.75 (2dp)
Specificity = 0.43 (2dp)


0.08

The post-test probability = post test odds/ (1 + post-test odds)

So first you'll need the post-test odds

Post-test odds = pre-test odds x likelihood ratio

But you'll need the pre-test odds too (stick with it, we're nearly there)

Pre-test odds = pre-test probability/ (1 pre-test probability)
= 1/16 / (1 - 1/16)
= 1/16 / 15/16
= 16 / 240
= 0.0667

So

Post-test odds = pre-test odds x likelihood ratio
= 0.0667 x (sensitivity / (1 - specificity))
= 0.0667 x (0.75 / (1 - 0.43))
= 0.0877

So

The post-test probability = post test odds/ (1 + post-test odds)
= 0.0877 / 1.08
= 0.081 (2dp)
