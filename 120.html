Which of the following best exemplifies concurrent validity in a psychiatric context?

Using a new depression scale and comparing its results to a validated scale administered simultaneously

The correct answer is Using a new depression scale and comparing its results to a validated scale administered simultaneously. Concurrent validity evaluates whether a new measure correlates well with an established measure of the same construct administered at the same time. This option directly tests the concept.

Testing a depression scale against an anxiety scale to assess differentiation: This tests discriminant validity, not concurrent validity. Discriminant validity assesses whether the scale measures a construct distinct from other unrelated constructs.

Administering a new schizophrenia scale and reassessing the same patients in six months: This evaluates test-retest reliability or predictive validity rather than concurrent validity, as it examines consistency or predictive capabilities over time.

Using a new personality inventory to predict future job performance: This relates to predictive validity, which assesses whether a measure accurately forecasts a future outcome or behaviour, not concurrent validity.

Comparing a new anxiety scale to a scale measuring physical health symptoms: This does not reflect concurrent validity because the comparison is between unrelated constructs. Concurrent validity requires the comparison of measures targeting the same or closely related constructs.
