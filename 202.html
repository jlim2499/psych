Regarding hypothesis testing. A type II error is also known by which term?

False negative

Type I error = alpha error, error of the first kind, or a false positive

Type II error = beta error, error of the second kind or the false negative
Explanation for the Correct Answer:

False negative is the correct term associated with a Type II error in hypothesis testing. A Type II error occurs when the test fails to reject the null hypothesis when in fact the alternative hypothesis is true, meaning a real effect or difference exists but the test incorrectly indicates that there is no effect or difference.

Explanation for the Incorrect Answers:

Error of the first kind is synonymous with a Type I error, not a Type II error. A Type I error occurs when the test incorrectly rejects a true null hypothesis.

True positive is not an error but a correct identification. It occurs when the test correctly identifies a true effect or difference.

Alpha error is another term for a Type I error, not a Type II error. It represents the probability of incorrectly rejecting the null hypothesis.

True negative is also not an error but a correct rejection. It occurs when the test correctly fails to reject the null hypothesis when there is no real effect or difference.
