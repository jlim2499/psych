In a randomised control trial assessing the efficacy of fluvoxamine in children with anxiety disorders, what type of data was used to measure outcomes on the Clinical Global Impressions Improvement scale?

Ordinal

Explanation for the Correct Answer:

Ordinal is the correct option. The Clinical Global Impressions Improvement scale measures improvement in a patient's clinical status and ranks it on a scale, typically from 1 (very much improved) to 7 (very much worse). This scale is inherently ordinal because each number represents a category that has a logical order in terms of severity or improvement but does not quantify the exact amount of improvement between each category.

Explanation for the Incorrect Answers:

Interval data would have equal distances between points on the scale, which is not the case with the Clinical Global Impressions Improvement scale as it does not measure the 'interval' between different levels of clinical status.

Dichotomous data represent variables that have only two categories or states (e.g., yes/no, true/false). The Clinical Global Impressions Improvement scale has more than two possible categories.

Numerical data imply that the data can be meaningfully added, subtracted, multiplied, or divided (i.e., quantitative data), but the Clinical Global Impressions Improvement scale is not quantitative.

Continuous data are measured on a continuum or a scale that has an infinite number of possible values within a selected range, which does not apply to the Clinical Global Impressions Improvement scale as it is a discrete scale with a finite number of categories.
