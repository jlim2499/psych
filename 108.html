Which of the following is the correct term for the above graph?


Kaplan-Meier curve

The correct answer is Kaplan-Meier curve. Kaplan-Meier curves are used in medical research to display the probability of a particular event (such as death, recurrence of disease, etc.) over time. The y-axis usually represents the survival function from 0 to 1 (or percentage from 0% to 100%) and the x-axis represents time. The graph shows step-like changes which occur at each event time.

The Stem and Leaf Plot is incorrect because this type of plot is a simple way to display data where each data value is split into a 'stem' (the first digit or digits) and a 'leaf' (the last digit). This kind of plot can provide information about the distribution, symmetry, skewness, outliers, and gaps in data but it does not represent survival probabilities over time.

The Galbraith plot, also known as radial plot or Galbraith's radial plot, is incorrect as well. This plot is used in meta-analysis for detecting heterogeneity across studies or identifying outlier studies. It plots standardised effect sizes against their precision.

A Scatter graph is also an incorrect option. Scatter graphs are used to demonstrate correlation between two variables where paired data are plotted on x and y axes. They do not show survival probabilities over time.

Finally, the Forest plot, another tool used in meta-analysis, is incorrect too. A forest plot displays individual study results together with an overall summary statistic representing the pooled result from all studies. It does not represent survival probabilities over time like a Kaplan-Meier curve does.
