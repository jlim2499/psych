Which of the following types of regression is used with dichotomous variables?

Logistic

The correct answer is Logistic regression. Logistic regression is a statistical method for analysing a dataset in which there are one or more independent variables that determine an outcome. The outcome is measured with a dichotomous variable (in which there are only two possible outcomes). In other words, it predicts the probability of occurrence of an event by fitting data to a logistic function. Hence, it is also known as logistic regression. Since its outcome is binary, it can be used to model the likelihood of a disease or health condition occurring.
