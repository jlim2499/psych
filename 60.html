Calculate the median value from the following values.
1, 3, 3, 3, 4, 5, 5, 6, 6, 6, 6


5

The median is the middle value. In this case there are 11 values so the point at which divides the set is 5. As you can see below, by identifying the median of 5 there are 5 values below and 5 values above it. Hence it divides the data set.

1,3, 3, 3, 4, 5, 5, 6, 6, 6, 6

The modal (most common value) value would be 6

The mean value would be 4.36 which is equal to the sum of all the values divided by the number of values.
