Which of the following correctly matches the data type with its characteristic?

Continuous data – infinite number of possible values within a range

The correct answer is Continuous data – infinite number of possible values within a range. Continuous data can take any value within a given range, such as height, weight, or temperature. These values are measurable and can include fractions or decimals.

Qualitative data – measured on an interval scale: This is incorrect. Qualitative data refers to non-numerical data describing characteristics or categories (e.g., gender, ethnicity), and it is not measured on an interval scale. Interval scales are used for numerical data.

Discrete data – values are not countable: This is incorrect. Discrete data involves countable values, such as the number of children in a household or the number of hospital admissions.

Nominal data – has a natural order: This is incorrect. Nominal data is purely categorical with no natural order, such as blood type or eye colour. Data with a natural order would be ordinal.

Ordinal data – purely categorical without any order: This is incorrect. Ordinal data is categorical but has a meaningful order, such as pain severity scales (e.g., mild, moderate, severe).
