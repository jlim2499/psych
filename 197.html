A study was conducted on a new antipsychotic medication to investigate if it resulted in akathisia. Following a wash out period (to enable elimination of any medication they might have been prescribed) patients with schizophrenia were randomly allocated to the new antipsychotic or to olanzapine. The patients were followed up over a 4 week period. At the end of the study period, each patient was assessed to see if they had experienced akathisia.

Considering the design of this study answer the following

237.What type of variable is the dependent variable?
Categorical The correct answer is categorical. The dependent variable in this study is whether or not the patient experienced akathisia. This is a categorical variable because the outcome is binary (yes/no). Categorical variables are used to represent distinct groups or categories, and in this case, the categories are 'akathisia present' and 'akathisia absent'.

ordinal: This would not be correct because an ordinal variable represents data with a meaningful order or ranking, such as severity levels (e.g., mild, moderate, severe). In this study, the outcome is binary without any ranking, making it categorical rather than ordinal.

ratio: This is incorrect because ratio variables represent continuous data with a true zero point, such as age, weight, or height. The experience of akathisia does not fit this definition, as it does not involve continuous numerical measurement.


238.The researcher proposes that the number of patients that will develop akathisia will not differ between the new medication and olanzapine. Which type of hypothesis is stated here?
Null hypothesis The correct answer is null hypothesis. The researcherâ€™s statement proposes that there is no difference in the number of patients developing akathisia between the new medication and olanzapine. A null hypothesis assumes no effect or no difference between groups and serves as the default assumption in statistical testing until evidence suggests otherwise.

research hypothesis: This is incorrect because a research hypothesis (or alternative hypothesis) suggests that there is an effect or difference between groups. For example, if the researcher hypothesised that more patients would develop akathisia with the new medication compared to olanzapine, it would represent a research hypothesis.


239.Which sample statistic would be most useful to address the null hypothesis?
Chi-squared test The correct answer is Chi-squared test statistic. The null hypothesis states that there is no difference in the proportion of patients developing akathisia between the new medication and olanzapine. Since the dependent variable (presence or absence of akathisia) is categorical and the independent variable (medication type) involves two independent groups, the chi-squared test is the most appropriate test statistic for this scenario.

Five steps to address the null hypothesis:

1) Define the research question: Determine whether there is a significant difference in the proportion of patients who develop akathisia between the two groups (new medication vs olanzapine).
2) Establish the groups: There are two independent groups: one receiving the new medication and the other receiving olanzapine.
3) Identify the variables and their data types: Dependent variable: Presence or absence of akathisia (categorical, binary). Independent variable: Medication type (categorical, with two levels).
4) Establish whether the data are parametric or non-parametric: The data are non-parametric because the dependent variable is categorical.
5) Select the appropriate statistical test: Use a chi-squared test statistic to compare the proportions of akathisia between the two groups.

Explanation of other options:

t-test: This is incorrect because a t-test compares means of continuous data between groups. In this study, the dependent variable is categorical (presence or absence of akathisia), so a t-test is not suitable.

odds ratio: While the odds ratio could be calculated to interpret the strength of the association between the medication type and akathisia, it is not a sample statistic directly used to test the null hypothesis. It is typically reported as part of the results.

regression coefficient: This is not appropriate because regression coefficients are used in models that predict a dependent variable based on one or more independent variables. This study does not involve a predictive model but rather a comparison between two groups.

ANOVA test statistic: This is incorrect because ANOVA is used to compare means of continuous data across three or more groups. In this case, the dependent variable is categorical, and there are only two groups being compared.
