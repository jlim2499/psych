What is the median value from the following data set?
1, 1, 2, 2, 3, 4, 5, 5, 5, 90


3.5

The median is the value at which half the values are below and half above.

1, 1, 2, 2, 3, 4, 5, 5, 5, 90

In this case there are an even number of values so the median is half way between the two middle values (3 and 4) so it is 3.5.

The modal value (most common) is 5 and the mean is 32.5 (sum of all values/ number of values which is 325/10= 32.5).

This is a good example to illustrate how the mean is sensitive to outliers whereas the median is not. Note how in this case a value of 32.5 does not give a good indication of an average for this range of values. This is due to the outlier of 90 which distorts the mean.
