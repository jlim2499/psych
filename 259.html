A new blood test is developed to diagnose early onset Alzheimer's disease. When compared to the gold standard in a study the following results are obtained.

True positive = 80
True negative = 10
False positive = 30
False negative = 40

Based on the above results calculate the following

319.Sensitivity of the new test
0.67 Sensitivity Calculation

Formula: Sensitivity = (True Positives) / (True Positives + False Negatives)

Values: True Positives (TP) = 80 False Negatives (FN) = 40

Calculation: Sensitivity = TP / (TP + FN) Sensitivity = 80 / (80 + 40) Sensitivity = 80 / 120 Sensitivity = 0.67 (or 67%)

Conclusion: The sensitivity of the new test is 67%. This means that 67% of patients with Alzheimer's disease were correctly identified by the new test.


320.Likelihood ratio for a positive test result
0.89 Likelihood Ratio for a Positive Test (LR+) Calculation

Formula: LR+ = Sensitivity / (1 - Specificity)

Step 1: Calculate Sensitivity Sensitivity = True Positives / (True Positives + False Negatives)
Sensitivity = 80 / (80 + 40) = 0.67

Step 2: Calculate Specificity Specificity = True Negatives / (True Negatives + False Positives)
Specificity = 10 / (10 + 30) = 0.25

Step 3: Calculate LR+ LR+ = Sensitivity / (1 - Specificity)
LR+ = 0.67 / (1 - 0.25)
LR+ = 0.67 / 0.75
LR+ = 0.89

Interpretation:

LR+ > 10: Strong evidence to rule in the disease.
LR+ between 5-10: Moderate evidence to rule in the disease.
LR+ between 2-5: Weak evidence to rule in the disease.
LR+ between 1-2: Very weak evidence to rule in the disease.
LR+ = 1: No diagnostic value.
LR+ < 1: The test result is more likely to occur in individuals without the disease than with the disease.

Conclusion: The LR+ for this test is 0.89, which suggests that the test does not provide useful diagnostic information for ruling in Alzheimer's disease. A result less than 1 means that a positive test result is slightly more likely to occur in someone without the disease than in someone with the disease.


321.Likelihood ratio for a negative test result
1.33 Likelihood Ratio for a Negative Test (LR-) Calculation

Formula: LR- = (1 - Sensitivity) / Specificity

Step 1: Calculate Sensitivity Sensitivity = True Positives / (True Positives + False Negatives)
Sensitivity = 80 / (80 + 40) = 0.67

Step 2: Calculate Specificity Specificity = True Negatives / (True Negatives + False Positives)
Specificity = 10 / (10 + 30) = 0.25

Step 3: Calculate LR- LR- = (1 - Sensitivity) / Specificity
LR- = (1 - 0.67) / 0.25
LR- = 0.33 / 0.25
LR- = 1.33

Interpretation:

LR- < 0.1: Strong evidence to rule out the disease.
LR- between 0.1 - 0.2: Moderate evidence to rule out the disease.
LR- between 0.2 - 0.5: Weak evidence to rule out the disease.
LR- between 0.5 - 1: Very weak evidence to rule out the disease.
LR- = 1: No diagnostic value.
LR- > 1: The test result is more likely to occur in individuals with the disease than without the disease.

Conclusion: The LR- for this test is 1.33, which means that a negative test result is slightly more likely to occur in someone with Alzheimer's disease than in someone without it. Therefore, this test is not useful for ruling out the disease.
